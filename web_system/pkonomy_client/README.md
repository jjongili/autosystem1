# 프코노미 클라이언트

시스템 트레이에 상주하면서 웹서버와 연동하는 클라이언트 프로그램입니다.

## 기능

1. **자동 로그인** - 웹서버 요청 시 자동 로그인 (크롬 디버깅 모드, 세션 유지)
2. **수동 로그인** - 직접 플랫폼/아이디/비밀번호 입력하여 로그인
3. **올인원 작업** - 중복삭제, 품절체크, 원가반영 등
4. **알리 수집** - 알리익스프레스 송장번호 수집

## 설치

```bash
# 1. 의존성 설치
pip install -r requirements.txt

# 또는 install.bat 실행
```

## 실행

```bash
python client.py

# 또는 run.bat 실행
```

## 설정

프로그램 실행 후 "설정" 탭에서:
- 서버 URL: 웹서버 주소 (기본: http://localhost:5000)
- API 키: 인증 키
- 크롬 경로: Chrome 실행 파일 경로
- 디버그 포트: 크롬 디버깅 포트 (기본: 9222)

## 크롬 디버깅 모드

- 각 로그인 세션은 별도 크롬 프로필로 실행됩니다
- 세션이 유지되어 재로그인 필요 없음
- 프로필 저장 위치: C:\autosystem\chrome_profiles\

## 시스템 트레이

- 창을 닫아도 트레이에서 계속 실행
- 트레이 아이콘 더블클릭으로 창 열기
- 우클릭 메뉴에서 종료
